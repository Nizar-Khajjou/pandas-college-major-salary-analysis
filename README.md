# College Major Salary Analysis with Pandas

A data exploration project analyzing post-graduation salary outcomes across 50 college majors, using a dataset from PayScale Inc.'s survey of 1.2 million Americans with bachelor's degrees.

## Key questions explored
- Which degrees have the highest starting salaries?
- Which majors have the lowest earnings after college?
- Which degrees have the highest earning potential?
- Which majors carry the lowest earnings risk (smallest gap between low and high earners)?
- Do Business, STEM, or HASS (Humanities, Arts, Social Science) degrees earn more on average?

## Key finding
Grouping majors into Business, STEM, and HASS categories revealed STEM degrees have the highest mid-career median salary ($90,812.50), while HASS degrees have the lowest ($62,968.18) — though individual majors within each category vary significantly.
